\# GameParser



Console application built in C# while learning Object-Oriented Programming.



\## Features



\- Validates user input

\- Reads JSON files from disk

\- Parses and displays video game data

\- Logs exceptions to a file



\## Technologies



\- C#

\- .NET

\- Object-Oriented Programming



\## Purpose



This project was created while learning C# and practicing OOP concepts such as classes, exception handling, file reading and JSON parsing.
