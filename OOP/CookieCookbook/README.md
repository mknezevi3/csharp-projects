\# Cookie Cookbook



Console application built in C# while learning Object-Oriented Programming.



\## Features



\- Add new recipes

\- Display recipes

\- Save recipes to file

\- Load recipes from file



\## Technologies



\- C#

\- .NET

\- Object-Oriented Programming



\## Purpose



This project was created while learning C# and practicing OOP concepts such as classes, lists and file handling.
